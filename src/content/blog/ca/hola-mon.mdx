Benvingut al nou sistema de blog. Aquest és un post de prova en català.

## Benvingut al Nou Sistema

Aquest és un sistema de blog completament funcional construït amb Astro i MDX. Permet crear contingut ric amb components integrats i suport complet per a múltiples idiomes.

### Característiques principals

- **Suport multiidioma**: Funciona perfectament en català, espanyol i anglès
- **Contingut MDX**: Pots escriure contingut ric amb components
- **Responsive**: S'adapta a tots els dispositius
- **SEO optimitzat**: Títols i metadades correctes

## MDX

Pots escriure contingut en MDX i incloure components si ho necessites. El sistema està dissenyat per ser flexible i fàcil d'usar.

### Estructura de carpetes

El sistema de blog està organitzat de manera molt senzilla:

```
src/content/blog/
├── ca/           # Posts en català
│   └── hola-mon.mdx
├── en/           # Posts en anglès
│   └── hello-world.mdx
└── es/           # Posts en espanyol
    └── hola-mundo.mdx
```

### Configuració d'idiomes

Cada post té metadades de traducció que permeten la navegació entre idiomes:

```javascript
// En el frontmatter de cada post
---
title: "Hola Món"
date: 2025-10-05
excerpt: "Entrada de prova en català"
translations:
  es: "hola-mundo"  # Slug del post en espanyol
  en: "hello-world" # Slug del post en anglès
---
```

### Sistema de rutes

Les rutes es generen automàticament basant-se en l'estructura de carpetes:

```javascript
// Rutes generades automàticament
/ca/blog/hola-mon     // Post en català
/es/blog/hola-mundo   // Post en espanyol
/en/blog/hello-world  // Post en anglès
/ca/blog             // Llista de posts en català
/es/blog             // Llista de posts en espanyol
/en/blog             // Llista de posts en anglès
```

### Components i estils

El sistema utilitza components reutilitzables i estils consistents:

```javascript
// Component de blog card
<BlogCard 
  title={post.data.title}
  excerpt={post.data.excerpt}
  date={post.data.date}
  image={post.data.image}
  slug={post.slug}
/>
```

### Funcionalitats principals

- **Multiidioma**: Suport complet per a català, espanyol i anglès
- **MDX**: Contingut ric amb components integrats
- **Responsive**: Adaptació automàtica a tots els dispositius
- **SEO**: Metadades optimitzades per a motors de cerca
- **Navegació**: Sistema de traducció automàtica entre posts

Aquest és només el començament del que pots fer amb aquest sistema de blog.
